WaterSmart Help Site

Building the WaterSmart Software help site helped me to understand how research and testing can dramatically impact user flow based on their hierarchy of needs. It was necessary to create a site that is easy to navigate, search friendly, and contains easy step-by-step instructions with screenshots and gifs for further explanation. When testing out prototypes, the team noted that users first looked for the search bar when helping a customer. This gave them quick results that enabled them to help end users faster. It was key to design the site so the first step was to find the search bar to quickly help answer customer questions and than create a category area for further exploration.
